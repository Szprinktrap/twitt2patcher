Don't fret, here's how to deal with it.

Background

Since Android 14, Google introduced this completely ridiculous, unnecessary, dumb and annoying "feature" called "Low-Target SDK Block".
What it does is preventing apps targetting Android version 5 (Lolipop) and lower from being installed using "normal" means. It's completely stupid and I have no idea what Google was smoking while coming up with it. Probably the same shit they were smoking when coming up with Play Integrity/SafetyNet.

The good news is that this block can be bypassed, both using root and not using root!
I'll show you 3 methods to do this, from (I think) easiest to hardest.

Method 1 (root): LSPosed and Disable Target API Block module.

You need to be rooted to do this.
  1. Install LSPosed, if you haven't already. If it doesn't work, try using this mod, it's what I've been using.
  2. Install the Disable Target API Block module. WARNING: If you have the useless Play Protect feature enabled, it WILL try to scare you that it's malicious! Tap on "Show more" and install it anyway.
  3. Enable the module for recommended apps. You should get a notification from LSPosed once you install the APK.
  4. Reboot your device.
  5. Install the patched APK as you would normally.
  6. Enjoy!

Method 2 (no root): Shizuku and "Install With Options" app.

You don't need root to do this, though for setting up Shizuku you need to either be connected to Wi-Fi or have a PC with ADB ready.
  1. Install Shizuku, if you haven't already.
  2. Set up Shizuku by following the instructions in the app. If needed, you also might refer to Shizuku's User Guide.
  3. Install the "Install With Options" app.
  4. In the app, check the "Bypass Low-Target SDK Block" option.
  5. Tap on Select files, select your patched Twitter APK and hit Install!
  6. Enjoy!

Method 3 (no root): ADB.

You don't need root to do this, however you'll need a PC with ADB (and proper drivers).
  1. If you don't have it on your PC, transfer the patched APK to it.
  2. Open up a terminal, command prompt, PowerShell, whatever the fuck you use for l33t h4xx0rring.
  3. In the terminal, navigate to the directory where your APK is, if you aren't in it already.
  4. Run adb install --bypass-low-target-sdk-block your-patched-apk-name.apk
  5. Wait for the Success message to appear.
  6. Enjoy!
